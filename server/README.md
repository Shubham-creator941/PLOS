# Personal Learning Operating System (PLOS) - Backend

Backend service for the AI-powered Personal Learning Agent.

## Tech Stack
- Node.js & Express.js
- TypeScript
- MySQL 8 (via `mysql2`)
- JWT Authentication

## Setup Instructions
1. Install dependencies: `npm install`
2. Configure environment: Copy `.env.example` to `.env` and fill in credentials.
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
